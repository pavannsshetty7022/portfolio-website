Connecting Citizens with Governance

JanDrishti – Community Issue Tracker

Built a centralized system where citizens can report real-world problems like potholes, water shortages, waste issues, and streetlight failures, enabling transparent tracking and faster resolution through real-time communication.

Key Features
  • Community-driven reporting for civic issues
  • Real-time issue status updates using WebSockets
  • Role-based access for citizens, admins, and authorities using JWT
  • Categorization by location, issue type, and priority
  • Responsive and offline-first design for low-connectivity environments
Use Case

Designed for municipalities, colleges, gated communities, and local bodies to improve accountability and complaint resolution.
